# Ruins Plugin

Plugin officiel du serveur **Ruins** pour Minecraft **Spigot 1.20.1**.

Ce plugin regroupe plusieurs systèmes essentiels du serveur comme :
- authentification personnalisée
- gestion des joueurs
- système de shop
- écran de ban personnalisé
- stockage des données

## 1. Présentation

Le Ruins Plugin est un plugin Spigot développé spécifiquement pour le serveur Ruins.

Il centralise plusieurs fonctionnalités importantes du serveur :
- authentification pour joueurs crack
- gestion des données joueurs
- ecran de ban personnalisé
- shop

Toutes les données sont stockées dans MongoDB afin de garantir de bonnes performances et une bonne scalabilité.

## 2. Technologies utilisées

Le projet utilise les technologies suivantes :

- **Java 17**
- **Spigot API 1.20.1**
- **MongoDB**
